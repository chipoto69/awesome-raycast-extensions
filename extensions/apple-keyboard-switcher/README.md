# Apple Keyboard Switcher

Quickly disconnect your Apple Magic Keyboard from the current Mac so it connects to your other computer.

## Requirements

Install `blueutil`:

```bash
brew install blueutil
```

## Usage

1. Search for "Disconnect Apple Keyboard" in Raycast
2. Select your keyboard and choose **Disconnect Keyboard**
3. The keyboard will immediately connect to your other Mac

## Hotkey Recommendation

Assign a global hotkey (e.g. `⌘ + K`) to the command for instant switching.
