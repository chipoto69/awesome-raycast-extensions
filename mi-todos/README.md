# MiToDos

Markdown-based todo manager that lives in your Obsidian wiki.

## Commands

| Command | Description |
|---|---|
| `Add Task` | Append a task to `mitodos.md` inbox |
| `Create Project` | Create `mitodos-<project>.md` in wiki `_meta/` |
| `Search MiToDos` | Semantic search across todos and wiki with QMD |

## Setup

1. Set your wiki path in preferences (default: `~/wiki`)
2. Set your mitodos file path (default: `~/wiki/_meta/mitodos.md`)
3. Optional: Install [QMD](https://github.com/garrytan/qmd) for semantic search — grep fallback is automatic

No separate database. No API. Just markdown in your wiki.
